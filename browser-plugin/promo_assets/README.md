# Chrome Web Store Promotional Assets

This directory contains the promotional images required for submitting the extension to the Chrome Web Store. The images currently here are placeholders with the correct dimensions.

Dashboard: <https://chrome.google.com/webstore/developer/dashboard>

## Image Specifications

### Store Icon
- **Filename:** `store_icon_128x128.png`
- **Dimensions:** 128 x 128 pixels
- **Format:** PNG
- **Description:** This is the main icon for your extension in the Chrome Web Store.

### Screenshots
- **Filenames:** `screenshot_1_1280x800.png`, `screenshot_2_1280x800.png`
- **Dimensions:** 1280 x 800 pixels (or 640 x 400)
- **Format:** JPEG or 24-bit PNG (no alpha transparency)
- **Description:** You can have up to five screenshots to showcase your extension's functionality. At least one is required.

### Small Promotional Tile
- **Filename:** `small_promo_tile_440x280.png`
- **Dimensions:** 440 x 280 pixels
- **Format:** JPEG or 24-bit PNG (no alpha transparency)
- **Description:** This tile is used for promotional purposes in various locations within the Chrome Web Store.

### Marquee Promotional Tile
- **Filename:** `marquee_promo_tile_1400x560.png`
- **Dimensions:** 1400 x 560 pixels
- **Format:** JPEG or 24-bit PNG (no alpha transparency)
- **Description:** This is a larger promotional tile used for featuring in the Chrome Web Store's marquee carousel.
